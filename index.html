# 上海市宝山区长江路780弄848弄淞南二村小区建筑垃圾堆点记录

# 2020年8月左右，物业在上海市宝山区长江路780弄21号居民楼外墙上，插了建筑垃圾临时堆放点的告示牌。

# 起初以为是本居民楼内有人装修才临时树立，但是时间过去了一个多月，告示牌还是继续树立在此，建筑垃圾也持续不断增多，影响居民楼环境和居民正常出行。

# 之后通过上海市市民服务热线12345多次反映，回复中表示此处已被设立成了长江路780弄848弄淞南二村小区唯一的建筑垃圾堆放点。然后本人已经50多次反映此处堆放点影响居民正常通行和生活，以及造成污染环境。

# 但是管理部门，淞南镇房屋管理事务所，长江路第一居民委员会和淞南镇环卫所，完全不予理会，继续将建筑垃圾临时堆放点常年设立在此。如今依旧设立在那。

# 2021年2月开始已经3次向淞南镇信访反映，但是长江路第一居民委员会每次都以没有其他地方堆放和已经过小区业委会同意为由不肯搬离。但是事实上是，本居民楼也没有地方堆放建筑垃圾。而且小区其他地方有很多比我们居民楼还宽敞很多的空地应该可以用来堆放建筑垃圾，但是居委完全无视。

# 另外，此处居民并没有同意过建筑垃圾可以堆放在此，当居民通过信访问居委会要业委会联系地址和方式时，居委会表示业委会没有联系方式，拒不提供，还说什么在改选等，拖延的话来敷衍居民。

# 此处堆放点也是780弄21号居民的外墙，也是21号居民出行的必经之路，此处堆放和清理建筑垃圾产生的噪音和震动每天影响着我们，所以我们没有办法妥协，一定会用尽所有途径投诉包括起诉等手段维权到底。

# 小区一共有50栋居民楼，现在唯一的想法就是可以恢复原状。以前都是谁装修就堆放在谁家楼下，附近大多数小区现在也是如此，这样可以大大减少对其中一栋居民影响。

# 2021年11月宝山区信访联系了居委会，物业，和业委会一起商讨此事，但是很遗憾没有得到合理的解决，建筑垃圾依旧堆放在此，并表示只能通过业主起诉业委会来解决此事。并且表示小区需要设立建筑垃圾堆放点是上海市建筑垃圾管理规定中的规定。另外由于上海市垃圾分类的原因，垃圾房内不能堆放建筑垃圾等其他垃圾也是镇府的规定。

# 对于一个以前没有建筑垃圾和其他垃圾堆放点的小区。以前有些垃圾还能堆放在垃圾房内，建筑垃圾也可以分散堆放，现在却只能集中的堆放在780弄21号外墙上。

# 对于宝山区政府的决定还是非常失望的。后续只能希望宝山区法院，或者上海市法院给出一个公正的判决了。
