# 上海市宝山区长江路780弄848弄淞南二村小区建筑垃圾堆点记录

# 2020年8月左右，物业在上海市宝山区长江路780弄21号居民楼楼下，插了建筑垃圾临时堆放点的告示牌。

# 从那天开始噩梦就开始了，物业每天会将整个小区50多栋居民楼，将近1200户居民的所有装修垃圾，废旧家具都堆放在此。

# 由于堆放点是21号居民的日常出行的毕竟之路，所以对居民的日常出行造成了严重的影响。

# 然后每周又会用铲车铲除0-4次，不过铲除并没有什么实际效果，很快就会堆积的更多更高，而且由于建筑垃圾直接堆放在居民楼外墙，铲除时产生的噪音和震动，严重影响楼内居民的正常生活。

# 投诉履历

# 通过12345，信访，110投诉将近50多次，但是都被居委会以，经过业委会同意这一理由驳回。

# 由于居委会和物业以，业委会没有联系地址和联系方式为由，不肯告诉其他业主业委会的联系方式和地址，所以业主根本无法直接联系到业委会。

# 另外由于无法联系业委会，业委会信息不公开，起诉异常艰难，但是仍在努力中。
