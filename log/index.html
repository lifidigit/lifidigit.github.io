# 投诉履历

# 管理部门的回复结果大多都是让居民联系业委会，但是却拒绝提供业委会的联系方式，让居民只能无功而返。居民都不知道小区有业委会。信访回复中都写了居委，物业，业委会，环卫一起参与了设立，那为什么居委会不能提供业委会的联系方式？为什么居委，物业，环卫不能来解决这个问题？

# 询问业委会联系方式的信访答复没有被公开到网上，大致回复是，小区业委会暂时没有联系方式和联系地址。

# 信访回复中回复的加强管理，由于堆放点位设置不合理（堆放点太小，贴近居民出行过道，完全敞开），所以管理并没有什么效果。

# 向上海市12345投诉了将近50多次，但是由于是管理部门和居委会自己进行的回复，所以完全没有得到解决。已经放弃此投诉通道。

# 向宝山区信访反映也得到了，是业委会设立的，需要自行通过法院进行起诉的回复。

# 信访渠道也基本放弃，只能通过法律手段起诉。但是非常搞不明白，建筑垃圾管理规定也是政府制定的，垃圾分类规定也是政府制定的，那到底是起诉政府还是业委会。。。
